var variables can be updated and re-declared, let variables can be updated but not re-declared and const variables can neither be updated nor re-declared

We can use the new keyword on the regular function to create a new object.In a regular function, its internal this value is dynamic, it depends on how the function is invoked. On the other hand, In the arrow function, there are no this, if we access this in the arrow function it will return the this of the closest non-arrow parent function.

map() executes the same code on every element in an array and returns a new array with the updated elements, filter() checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria and forEach(), is used to execute the same code on every element in an array but does not change the array and it returns undefined.
